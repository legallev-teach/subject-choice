How to get a pre-filled link from your Google Form

  1. Open your Google Form in Edit mode.
  2. Click the three vertical dots (⋮) in the upper right corner.
  3. Select Get pre-filled link.
  4. Fill in every single field with any placeholder text — it doesn't matter what you type, as long as nothing is left blank.
  5. For the additional subjects section, tick both checkboxes (Religious Education and French).
  6. Click Get link at the bottom, then Copy link.
  7. Send that link to the website maintainer — they will update the HTML file and re-upload it to the server.

Why tick both checkboxes?

Google Forms handles checkbox fields differently from other question types. If only one checkbox is ticked when the pre-filled link is generated, the field may not be identified correctly when the HTML file is updated. Always tick both to be safe.

How to open this panel

  • Desktop: Press Ctrl + Shift + A anywhere on the form page.
  • Phone or tablet: Tap the school name in the blue header five times quickly (within three seconds).

Closing the form immediately — use Google Forms

This is the fastest way and you can do it yourself without any technical help.

  1. Open your Google Form in Edit mode.
  2. Click the Responses tab at the top.
  3. Toggle Accepting responses off.
  4. Students who try to submit after this point will see a "no longer accepting responses" message from Google directly.

To reopen: go back to the same toggle and switch it on again.

Closing the form on the school website

This replaces the form with a custom closed message on the school website page. It requires the website maintainer.

  1. Contact the website maintainer (see the Maintainer tab for what they need to do).
  2. Tell them the academic year is closed and provide the message you want students to see.
  3. They will update the HTML file and re-upload it. Students will then see your message instead of the form.

Connecting a new Google Form each year

  1. Create your new Google Form for the upcoming year (or make a copy of the previous one).
  2. Follow the steps in the Google Form tab to generate a pre-filled link.
  3. Use the verify tool in that tab to confirm all fields are present.
  4. Send the verified pre-filled link to the website maintainer.
  5. The maintainer will update the HTML file and re-upload it to the server.

Summary — who does what

  • Principal: manages the Google Form (create, copy, close responses, get pre-filled link, verify it, send to maintainer)
  • Website maintainer: updates the HTML file and uploads it to the school website

Opening and closing the form

Find this line near the top of the HTML file:

const FORM_OPEN = true;

Change true to false to close the form. Change it back to true to reopen it. Save the file and re-upload it to the server.

You can also customise the message students see when the form is closed:

const CLOSED_MESSAGE = "Subject selection is now closed...";

Updating the academic year

Find this line and update it:

const ACADEMIC_YEAR = "2026–2027";

This updates the header, footer, and pledge text automatically.

Updating the Google Form connection

When the principal sends a new pre-filled link, parse it to extract the new GOOGLE_FORM_URL and ENTRY_IDS. The entry IDs are the entry.XXXXXXXXX values in the URL — each one maps to a field in the order listed in the ENTRY_IDS object. Use the verify tool in the Google Form tab to confirm the mapping before updating the file.

Update both constants in the source, save the file, and re-upload to the server.

Updating form classes

const FORM_CLASSES = [
  "3A - Amethyst",
  "3B - Beryl",
  "3C - Chrysolite",
];

Add, rename, or remove entries. Keep quotation marks and commas exactly as shown.

Updating subjects in a group

Find the GROUPS array. Each group looks like this:

{ num: 4, subjects: [
  { name: "Biology" },
  { name: "Food and Nutrition*", tech: true },
]},
  • Edit the name in quotes to rename a subject
  • Add a new line to add a subject
  • Delete a line to remove a subject
  • Technical subjects must have tech: true
  • If you rename a subject that appears in SUBJECT_FAMILIES, update it there too

Adding the school logo

Place the logo image file in the same folder as the HTML file, then find and update:

const LOGO_FILE = "cuc-logo.png";

Use a plain text editor

Always open the HTML file in a plain text editor — Notepad on Windows, TextEdit on Mac (in plain text mode). Never use Microsoft Word or Google Docs — they add invisible formatting that will break the file.

How the form works

  1. Student fills in personal details and makes subject selections on this page
  2. On submit, the school's Google Form opens in a new tab with all fields pre-filled
  3. Student reviews and clicks Submit on the Google Form
  4. Response is recorded in the Google Sheet as normal

Technical notes

  • Single HTML file — no server-side code, database, or installation required
  • Works in all modern browsers on desktop, phone, and tablet
  • No student data is stored in this file or on GitHub
  • The GitHub repository is public — it contains no sensitive credentials or student data
